# Hussain Lakhani Jewellers Website Plan

## Project Overview
Create a luxury, heritage-driven jewellery website for Hussain Lakhani Jewellers (established 1965, Karachi).

## Design Specifications
- **Color Palette**: Deep black (#0a0a0a), Ivory white (#faf9f6), Antique gold (#c9a227), Champagne (#d4c4a8)
- **Typography**: Elegant serif for headings (Playfair Display), Clean sans-serif for body (Montserrat/Lato)
- **Layout**: Minimal, spacious, editorial-style, mobile-first
- **Vibe**: Old-money, trust-first, premium, never salesy

## Page Sections
1. **Header/Navigation**: Logo, Collections, About, Reviews, Contact, WhatsApp CTA
2. **Hero Section**: "A Legacy of Pure Gold Since 1965" headline, subtext, dual CTAs
3. **Trust Signals**: 5.0⭐ Google Rating, "Serving Karachi since 1965", "Sarafa Bazaar, Mithadar"
4. **Collections**: Bridal Sets, Gold Jewellery, Custom Orders, Heritage Designs (full-width visuals)
5. **About Us**: 3-generation legacy story, personal and rooted tone
6. **Reviews**: Google reviews carousel with emphasis on trust/purity/family
7. **Visit Us**: Map embed, address, timings, WhatsApp & Call buttons
8. **Footer**: Quick links, contact info, social links, copyright

## Functionality
- Smooth scroll animations
- Mobile-responsive design
- WhatsApp integration (wa.me link)
- Google Maps embed
- Review carousel/slider
- Subtle hover effects and transitions
- SEO optimization with meta tags

## Technical Requirements
- Clean, semantic HTML5
- CSS3 with custom properties (variables)
- Vanilla JavaScript for interactions
- No external frameworks (keep it lightweight)
- Fast loading (optimized CSS/JS)
- SEO-friendly structure
